# TS React Starter
A lean React 19 + TypeScript starter kit.
Features:
• Functional React components with TypeScript support.
• Storybook 8 for building and previewing UI components.
• Node.js CI workflow via GitHub Actions for continuous testing.
• Jest for unit testing with coverage reports.
• ESLint and Prettier for code quality and formatting.
• Tailwind CSS integration for styling.
• Docker configuration for containerized development.
